# AI Platform Tools Workspace

## Purpose

This workspace owns all planning for model evaluation, prompt governance, training datasets, AI feedback loops, and policy implementation for AI capabilities in the CCC platform.

## Scope Rules

- Model evaluation and prompt governance must precede any AI capability going into production.
- Personal injury and KADOE are out of scope.
- Training datasets must comply with data governance and retention policy.
