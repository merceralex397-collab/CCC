# Unified Platform Workspace

## Purpose

This workspace owns planning for the mature end-to-end CCC platform, convergence roadmap, and the migration from Operational Core multi-tool state to a unified architecture. It coordinates convergence of parser, intake, storage, case workflow, evidence, valuation, communications, and AI capabilities into a single coherent platform.

## Scope Rules

- Work in this workspace is long-range planned until the Operational Core MVP (P1) is stable.
- All convergence planning requires active Operational Core tickets as input dependencies.
- Personal injury and KADOE are out of scope.
