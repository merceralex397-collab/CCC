# External Wording

## Prefer

- comparable retail market evidence
- relevant live market examples
- current replacement market
- evidence supportive of the assessed retail value
- current retail replacement market
- the selected examples are considered sufficiently comparable to the subject vehicle
- having regard to make, model, age, mileage, engine, transmission, specification and general condition

## Avoid

- cherry-picked
- selected to increase value
- highest adverts found
- client-favourable only
- we ignored lower adverts
- lower adverts were disregarded
- adverts chosen to beat the guide
- EVA
- Engineer Value
- Original Eng Value
- guide value
- guide valuation
- guide price
- uplift
