# Report Style

Write in a professional UK engineering valuation tone. Keep the logic easy to follow, neutral, and defensible under insurer, solicitor, or engineer scrutiny.

Use `DD/MM/YYYY` dates. Use `Our Ref` for the subject registration. Use `Your Ref` for the claim or matter reference. If a claim or matter reference is missing from the prompt and screenshots, ask for it before rendering the PDFs.

Format the assessed value with two decimals, for example `GBP 1,200.00` in text payloads and `£1,200.00` in rendered PDFs. Advert asking prices should remain as advertised and should not invent pence.

Round final values sensibly, normally to the nearest GBP 50, GBP 100, GBP 250, or GBP 500 depending on value and evidence strength.

Do not expose internal methodology wording. External PDFs must not mention the internal source system, internal guide figures, `Engineer Value`, `Original Eng Value`, `guide value`, `guide valuation`, `guide price`, or `uplift`. The report should say that the live examples are relevant comparable retail market evidence and support the engineer's assessed retail value where that is defensible.

The report and evidence pack should not materially differ between guide-supported and market-only work. Do not add headings, notes, labels, or commentary that disclose `valuation_mode`, missing guide values, or internal search strategy. Keep the same section order and present the same assessed retail value opinion.

Keep short subject-table fields short. For `Vehicle History`, use `No adverse history recorded` for a routine clean check. Only add extra wording where it records a material point such as an adverse marker, mileage discrepancy, import/export issue, write-off category, or another history caveat relevant to value.

Write valuation commentary in the same pattern as the reference report: identify the closest comparator, state why it is aligned to the subject vehicle, then explain how the wider market evidence supports the assessed retail value. Do not argue against the assessed figure in the commentary. Material differences can be acknowledged neutrally in advert notes, but avoid phrasing that undercuts the conclusion, such as "although it has lower mileage" or "but with much lower mileage" in the core valuation reasoning.
