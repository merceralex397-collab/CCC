# CE Visual Layout

Use this reference for branded Collision Engineers documents.

## Assets

- Logo: `src/skills/ce-house-style/assets/brand/logo.png`, `logo.svg`, and `logo.pdf`.
- Shared HTML templates: `src/skills/ce-house-style/assets/templates/_base.html.j2`, `report.html.j2`, `evidence_pack.html.j2`, and `styles.css`.
- Layout examples: `src/skills/ce-house-style/assets/feenoteexample.pdf`, `addendumreportexample.pdf`, and `totallossexample.pdf`.

## Page Rules

- A4 portrait unless the owning skill specifies otherwise.
- Use approximately 2 cm margins.
- Put the CE logo at top left.
- Put references and date at top right where the document format supports it.
- Use CE red `#C80A32` for section rules and report table accents.
- Use uppercase bold section headings with a red rule under the heading for formal reports.
- Use the footer text: `Collision Engineers Ltd | www.CollisionEngineers.co.uk | engineers@collisionengineers.co.uk`.
- Use the locked sans-serif stack `Arial, Helvetica, sans-serif`.
