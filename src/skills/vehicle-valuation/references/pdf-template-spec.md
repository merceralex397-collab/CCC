When to read this: read when editing PDF templates or diagnosing visual differences against the examples.

# PDF Template Specification

- Page: A4 portrait.
- Margins: approximately 18mm.
- Header: logo top left; reference/date block top right.
- Footer: repeat on every page with `Collision Engineers Ltd | www.CollisionEngineers.co.uk | engineers@collisionengineers.co.uk`.
- Titles: centred uppercase.
- Section headings: uppercase, bold, brand red rule beneath.
- Tables: red header row with white bold text, alternating row shading, compact padding.
- Currency cells: right aligned.
- Assessed value: highlighted bordered box and never split across pages.
- Links: `Open advert`, underlined, brand red, clickable.
- Overflow: repeat table headers on new pages and allow rows/comments to wrap without changing column layout.
