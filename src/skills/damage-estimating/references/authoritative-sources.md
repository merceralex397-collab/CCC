# Damage Estimating Authoritative Sources

Damage-estimating charges and doctrine are maintained in compact references plus source-backed detailed companions. The ABP source remains canonical for rates.

| Need | Canonical source |
| --- | --- |
| Labour rates, repair-plan fee, ADAS, EV/hybrid, specialist/admin charges, regional uplift | `src/skills/abp-rates/references/abp-2026-rates.md` |
| Estimating doctrine: failure modes, repair-vs-replace, impact-path propagation, completeness walkthrough | `src/skills/damage-estimating/references/estimate-framework.md`, `src/skills/damage-estimating/references/estimate-framework-full.md` |
| Paint time + material (AZT method, blends, surcharges) | `src/skills/damage-estimating/references/azt-paint-method.md` |
| Scrap / repairable category (A/B/S/N), MIAFTR/V5C | `src/skills/salvage-categorisation/references/abi-salvage-cop.md` |
| Domain vocabulary (Audatex/ABP/BS 10125 terms) | `ce-domain-glossary` |

When an ABP figure in an implementation note and `abp-rates` disagree, **`abp-rates` wins** because it is the version-managed source.
