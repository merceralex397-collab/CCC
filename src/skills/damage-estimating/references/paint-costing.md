# Paint Costing Mode

Use this mode for AZT-based paint time and material costing.

## Load

- `references/azt-paint-method.md`
- `references/atzpaintcalculation.pdf` only when the distilled method is insufficient.
- `src/skills/abp-rates/references/abp-2026-rates.md` for material-cost anchors.

## Workflow

1. Identify panels, substrate, paint system, and process type.
2. Apply blend propagation.
3. Add scratch-resistant clearcoat, two-colour, plastic, prep, and masking surcharges where applicable.
4. Return paint hours, material cost, blend-panel list, and assumptions.

This mode is assistive. The calling expert mode validates the result.
