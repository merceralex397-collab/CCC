# Rebuttal — authoritative sources & templates

Strengthen the 13-point diminution framework (`../projecttext.md`) with these sources, and use the bundled templates.

## Bundled templates (this folder)

- `diminution-report-template.docx` — the current CE diminution report template (structure, headings, Statement of Truth).
- `anticipated-arguments.docx` — the anticipated-arguments playbook (defendant arguments and expert responses).

## Sources to cite when applicable

| Argument | Source |
| --- | --- |
| Challenge specialist charges / regional uplift in the opposing estimate | `abp-rates` → `abp-2026-rates.md` |
| Challenge a salvage category decision (S/N, structural vs non-structural) | `salvage-categorisation` → `abi-salvage-cop.md` |
| Challenge a thin / under-quoted opposing estimate (named failure modes, completeness) | `repair-estimate` → `estimate-framework.md` |
| OEM constraint (e.g. reconditioned wheels not approved) relevant to damage characterisation | `manufacturer-standards` → `oem-index.md` |
| Correct terminology (diminution, Cat S/N, BS 10125, CPR Pt 35) | `ce-domain-glossary` |

Apply only the points/sources that genuinely fit the case — overstating weakens credibility (per `projecttext.md`).
