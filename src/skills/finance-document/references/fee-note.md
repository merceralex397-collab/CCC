# Fee Note Mode

Use for Collision Engineers expert-services fee notes.

## Structure

- CE house header and matter references.
- Addressee and date.
- Services rendered.
- Time/rate lines, expenses, disbursements if supplied.
- VAT treatment and total.
- Payment terms and footer.
